### Get Equity Order Update (Python)

Source: https://groww.in/trade-api/docs/python-sdk/feed

This Python code snippet demonstrates how to subscribe to equity order updates using the Groww API. It includes a callback function to process incoming data and provides options for both asynchronous (callback-based) and synchronous (polling) data retrieval. Ensure you have the 'growwapi' library installed and your API token is correctly configured. The function returns order update details.

```python
from growwapi import GrowwFeed, GrowwAPI

# Groww API Credentials (Replace with your actual credentials)
API_AUTH_TOKEN = "your_token"

groww = GrowwAPI(API_AUTH_TOKEN)
feed = GrowwFeed(groww)

def on_data_received(meta): # callback function which gets triggered when data is received
  if(meta["feed_type"] == "order_updates" and meta["segment"] == groww.SEGMENT_CASH):
    print(feed.get_equity_order_update())

feed.subscribe_equity_order_updates(on_data_received=on_data_received)

# This is a blocking call. Nothing after this will run.
feed.consume()

# OR
# you can also fetch data synchronously
feed.subscribe_equity_order_updates()
# order update can also be continuously polled using this method
for i in range(10):
    print(feed.get_equity_order_update())

feed.unsubscribe_equity_order_updates()
```

### Live Data for Equity and Derivatives

Source: https://groww.in/trade-api/docs/python-sdk/feed

Subscribe to, get, and unsubscribe from the live data for equities or derivatives contracts. This can be done asynchronously with a callback or synchronously by polling.

```APIDOC
## Live Data for Equity and Derivatives

### Description
Subscribe to, get, and unsubscribe from the live data for equities or derivatives contracts. This can be done asynchronously with a callback or synchronously by polling.

### Method
POST (for subscription/unsubscription)
GET (for polling live data)

### Endpoint
/feed/subscribe_ltp
/feed/get_ltp
/feed/unsubscribe_ltp

### Parameters
#### Request Body (for subscribe/unsubscribe)
- **instruments_list** (array) - Required - A list of instruments to subscribe to or unsubscribe from.
  - **exchange** (str) - Required - Stock Exchange (e.g., "NSE").
  - **segment** (str) - Required - Segment of the instrument (e.g., "CASH", "FNO").
  - **exchange_token** (str) - Required - Exchange token of the instrument.
- **on_data_received** (function) - Optional - Callback function triggered when data is received (for asynchronous usage).

### Request Example (Subscription - Asynchronous)
```python
from growwapi import GrowwFeed, GrowwAPI

API_AUTH_TOKEN = "your_token"
groww = GrowwAPI(API_AUTH_TOKEN)
feed = GrowwFeed(groww)

def on_data_received(meta):
    print("Data received")
    print(feed.get_ltp())

instruments_list = [{"exchange": "NSE", "segment": "CASH", "exchange_token": "2885"}, {"exchange": "NSE", "segment": "FNO", "exchange_token": "35241"}]

feed.subscribe_ltp(instruments_list, on_data_received=on_data_received)
feed.consume() # This is a blocking call
```

### Request Example (Subscription - Synchronous)
```python
from growwapi import GrowwFeed, GrowwAPI
import time

API_AUTH_TOKEN = "your_token"
groww = GrowwAPI(API_AUTH_TOKEN)
feed = GrowwFeed(groww)

instruments_list = [{"exchange": "NSE", "segment": "CASH", "exchange_token": "2885"}, {"exchange": "NSE", "segment": "FNO", "exchange_token": "35241"}]

feed.subscribe_ltp(instruments_list)

# Poll live data synchronously
for i in range(10):
  time.sleep(3)
  print(feed.get_ltp())

feed.unsubscribe_ltp(instruments_list)
```

### Response
#### Success Response (200)
- **ltp** (object) - Contains live traded price data for subscribed instruments.

        - **tsInMillis** (int) - Epoch time in milliseconds.
        - **ltp** (float) - The last traded price of the instrument.

### Response Example

  - **exchange_token** (str) - Required - Exchange token of the index (e.g., "NIFTY", "1").
- **on_data_received** (function) - Optional - Callback function triggered when data is received (for asynchronous usage).

### Request Example (Subscription - Asynchronous)
```python
from growwapi import GrowwFeed, GrowwAPI

API_AUTH_TOKEN = "your_token"
groww = GrowwAPI(API_AUTH_TOKEN)
feed = GrowwFeed(groww)

def on_data_received(meta):
    print("Data received")
    print(feed.get_index_value())

instruments_list = [{"exchange": "NSE", "segment": "CASH", "exchange_token": "NIFTY"}, {"exchange": "BSE", "segment": "CASH", "exchange_token": "1"}]
feed.subscribe_index_value(instruments_list, on_data_received=on_data_received)
feed.consume() # This is a blocking call
```

### Request Example (Subscription - Synchronous)
```python
from growwapi import GrowwFeed, GrowwAPI
import time

API_AUTH_TOKEN = "your_token"
groww = GrowwAPI(API_AUTH_TOKEN)
feed = GrowwFeed(groww)

instruments_list = [{"exchange": "NSE", "segment": "CASH", "exchange_token": "NIFTY"}, {"exchange": "BSE", "segment": "CASH", "exchange_token": "1"}]
feed.subscribe_index_value(instruments_list)

# Poll live data synchronously
for i in range(10):
    print(feed.get_index_value())

feed.unsubscribe_index_value(instruments_list)
```

### Get Greeks

```APIDOC
## GET /greeks

### Description

GET

### Subscribe to Derivatives Order Updates (Python)

Source: https://groww.in/trade-api/docs/python-sdk/feed

Enables asynchronous subscription to real-time order execution updates for derivatives. A callback function processes incoming updates, filtering by feed type and segment. Synchronous polling is also supported. Requires API authentication.

```python
from growwapi import GrowwFeed, GrowwAPI

# Groww API Credentials (Replace with your actual credentials)
API_AUTH_TOKEN = "your_token"

groww = GrowwAPI(API_AUTH_TOKEN)
feed = GrowwFeed(groww)

def on_data_received(meta): # callback function which gets triggered when data is received
  if(meta["feed_type"] == "order_updates" and meta["segment"] == groww.SEGMENT_FNO):
    print(feed.get_fno_order_update())

feed.subscribe_fno_order_updates(on_data_received=on_data_received)

# This is a blocking call. Nothing after this will run.
feed.consume()

# OR
# you can also fetch data synchronously
feed.subscribe_fno_order_updates()
# order update can also be continuously polled using this method
for i in range(10):
    print(feed.get_fno_order_update())

feed.unsubscribe_fno_order_updates()
```

### Groww Symbol Explanation

Source: https://groww.in/trade-api/docs/python-sdk/backtesting

Explanation of how Groww symbols are constructed for various instrument types (stocks, indices, futures, options).

```APIDOC
## Groww Symbol

Groww symbol is a unique identifier for an instrument across exchanges and segments. It is formed by concatenating:

* **Exchange**: Where the instrument is traded
* **Trading Symbol**: The name/ticker of the instrument
* **Expiry Date**: Only for derivatives (format: DDMmmYY, example: 23Jan25)
* **Strike Price**: Only for options (the target price level)
* **Option Type**: Only for derivatives:
    * CE = Call Option
    * PE = Put Option
    * FUT = Futures

**For Stocks and Indices:** Only exchange and trading symbol are used.
**For Futures:** Exchange, trading symbol, expiry date, and "FUT" are used.
**For Options:** All components are used including strike price and option type (CE/PE).

For example:
  * Equity: `NSE-WIPRO`, `BSE-RELIANCE`
  * Index: `NSE-NIFTY`, `BSE-SENSEX`
  * Future: `NSE-NIFTY-30Sep25-FUT`, `BSE-SENSEX-25Sep25-FUT`
  * Call Option: `NSE-NIFTY-30Sep25-24650-CE`, `BSE-SENSEX-25Sep25-79500-CE`
  * Put Option: `NSE-NIFTY-30Sep25-24650-PE`, `BSE-SENSEX-25Sep25-79500-PE`

Groww symbol also exists in the instruments csv file and it can be obtained from the Get Instruments API.

### Generate SHA-256 Checksum (Python, Java, .NET, JavaScript)

Source: https://groww.in/trade-api/docs/curl

Generates a SHA-256 checksum by concatenating a secret with a timestamp and hashing the result. This is a critical step for authenticating API requests. The function takes the API secret and the current epoch second timestamp as input and returns the hexadecimal representation of the SHA-256 hash. Ensure the timestamp is current, as it is valid for a limited time.

```python
import hashlib
import time

def generate_checksum(secret: str, timestamp :str) -> str:
    """
    Generates a SHA-256 checksum for the given data and salt.
    :param secret: The api secret value
    :return: Hexadecimal SHA-256 checksum
    """
    input_str = secret + timestamp
    sha256 = hashlib.sha256()
    sha256.update(input_str.encode('utf-8'))
    return sha256.hexdigest()

timestamp = int(time.time()) # Timestamp in epoch seconds
secret = "<Your secret here>"
checksum = generate_checksum(secret, str(timestamp))
print(checksum)
```

```java
import java.security.MessageDigest;
import java.security.NoSuchAlgorithmException;
import java.nio.charset.StandardCharsets;
import java.util.HexFormat;
import java.time.Instant;

public class ChecksumGenerator {

  private static final String SHA_256 = "SHA-256";

  public static void main(String[] args){
    System.out.println(generateChecksum("<your secret here>", getLatestTimestampinEpochSeconds()));
  }

  /**
   * Generates a SHA-256 checksum for the given input string.
   */
  public static String generateChecksum(String secret, String timestamp) {
    try {
      String input = secret + timestamp;
      MessageDigest digest = MessageDigest.getInstance(SHA_256);
      byte[] hash = digest.digest(input.getBytes(StandardCharsets.UTF_8));
      return HexFormat.of().formatHex(hash);
    } catch (NoSuchAlgorithmException e) {
      throw new RuntimeException("SHA-256 algorithm not found", e);
    }
  }

  /**
   * Return the latest time in epoch Seconds in String
   */
  public static String getLatestTimestampinEpochSeconds(){
    return String.format("%d", Instant.now().getEpochSecond());
  }
}
```

### Python
```python
import hashlib
import time

def generate_checksum(secret: str, timestamp :str) -> str:
    """
    Generates a SHA-256 checksum for the given data and salt.
    :param secret: The api secret value
    :return: Hexadecimal SHA-256 checksum
    """
    input_str = secret + timestamp
    sha256 = hashlib.sha256()
    sha256.update(input_str.encode('utf-8'))
    return sha256.hexdigest()

timestamp = int(time.time()) # Timestamp in epoch seconds
secret = "<Your secret here>"
checksum = generate_checksum(secret, str(timestamp))
print(checksum)

```java
import java.security.MessageDigest;
import java.security.NoSuchAlgorithmException;
import java.nio.charset.StandardCharsets;
import java.util.HexFormat;
import java.time.Instant;

public class ChecksumGenerator {

  private static final String SHA_256 = "SHA-256";

  public static void main(String[] args){
    System.out.println(generateChecksum("<your secret here>", getLatestTimestampinEpochSeconds()));
  }

  /**
   * Generates a SHA-256 checksum for the given input string.
   */
  public static String generateChecksum(String secret, String timestamp) {
    try {
      String input = secret + timestamp;
      MessageDigest digest = MessageDigest.getInstance(SHA_256);
      byte[] hash = digest.digest(input.getBytes(StandardCharsets.UTF_8));
      return HexFormat.of().formatHex(hash);
    } catch (NoSuchAlgorithmException e) {
      throw new RuntimeException("SHA-256 algorithm not found", e);
    }
  }

  /**
   * Return the latest time in epoch Seconds in String
   */
  public static String getLatestTimestampinEpochSeconds(){
    return String.format("%d", Instant.now().getEpochSecond());
  }
}
